Klotski

Klotski 是一个方便的 CSS Grid 框架，并支持现代 CSS Grid 在 IE11 的应用

基本用法

直接调用

直接调用 toGridProps 与 toItemProps 进行开发, 是最简单的调用方式

A
E
F
C
const template = `
  "A   B    C"  80px
  "E   F    G"  80px /
  100px 100px 100px
`;
const classes = makeStyles(() => ({
  root: {
    '& > div': {
      boxShadow: '0 0 0 1px #000 inset',
      lineHeight: '80px',
      textAlign: 'center',
    },
    ...toGridProps({
      inline: true,
      grid: template,
    }),
  },
  itemA: {
    ...toItemProps({
      $grid: template,
      gridArea: 'A'
    }),
  },
  itemE: {
    ...toItemProps({
      $grid: template,
      gridArea: 'E'
    }),
  },
  itemF: {
    ...toItemProps({
      $grid: template,
      gridArea: 'F'
    }),
  },
  itemC: {
    ...toItemProps({
      $grid: template,
      gridArea: 'C'
    }),
  },
}))({});
return (
  <div className={classes.root}>
    <div className={classes.itemA}>A</div>
    <div className={classes.itemE}>E</div>
    <div className={classes.itemF}>F</div>
    <div className={classes.itemC}>C</div>
  </div>
);

同时生成子父模板

使用 makeBundled 方法, 可以同时生成隐式包含模板信息的 toGridProps 与 toItemProps, 并且同时支持在解构下进行重命名

    ...item({
      gridArea: 'E'

    ...item({
      gridArea: 'F'

const template = `
  "A   B    C"  80px
  "E   F    G"  80px /
  100px 100px 100px
`;
// 同上
const map = [
  [
    ["A", "B", "C", '80px'],
    ["E", "F", "G", '80px']
  ],
  ['100px', '100px', '100px']
];
// 在批量模板模式下, grid 模板必须是字符串,后续会开放数组形式的模板生成器，
// 但是不论如何，grid必须是静态的, 去除的栅格生成器名为 `命名空间名（如果有）网格名`
// 就像下例中 生成器名就为 `item(命名空间)A(具体网格名)`
// 可以使用命名空间来对生成器重新命名来防止冲突，
// 但是不论命名空间如何变化它占用的栅格名是不会变化的
const [root, { itemA, itemB, itemC, itemE, itemF, itemG }] = makeBundled({
  $map: map,
}, {
  multi: true,
  itemNameSpace: 'item'
});
// 可以对比下在上面的例子中节省了多少代码
// const [root, { A: itemA, B: itemB , C: itemC, ... }] = makeBundled({
//   grid: template,
// }, {
//   multi: true,
// });
const classes = makeStyles(() => ({
  root: {
    '& > div': {
      boxShadow: '0 0 0 1px #000 inset',
      lineHeight: '80px',
      textAlign: 'center',
    },
    ...root({
      inline: true,
    }),
  },
  itemA: {
    ...itemA({}),
  },
  itemB: {
    ...itemB({}),
  },
  itemC: {
    ...itemC({}),
  },
  itemE: {
    ...itemE({}),
  },
  itemF: {
    ...itemF({}),
  },
  itemG: {
    ...itemG({}),
  },
}))({});
return (
  <div className={classes.root}>
    <div className={classes.itemA}>A</div>
    <div className={classes.itemB}>B</div>
    <div className={classes.itemE}>E</div>
    <div className={classes.itemF}>F</div>
    <div className={classes.itemC}>C</div>
    <div className={classes.itemG}>G</div>
  </div>
);

格栅模板

字符串式

网格模板在直接使用上相对具名模板并没有太大的优势，但是他作为具名模板的补充，可以进行更加精细与复杂的应用，并且由于网格模板的语义与 ie 中使用的 legacy grid 在语义上更为接近，所以可以用更少的转换以获得更高的性能

const template = '80px 80px 80px/ 100px 100px 100px'

高阶用法

跨网格

当在使用中，难免会需要一个元素同时占用多个 grid 容器,这时就可以使用跨容器 gridArea 来进行布局,而且也可以利用网格可以相互叠加的特定来做一些特殊的布局,比如下例中的外边框就是用两个元素同时叠加的

headbodyside
const template = `
  "A   B    C"  80px
  "D   E    F"  80px
  "G   H    I"  80px /
  100px 100px 100px
`;
const [root, item] = makeBundled({
  grid: template,
});
const classes = makeStyles(() => ({
  root: {
    '& > div': {
      boxShadow: '0 0 0 1px #000 inset',
      lineHeight: '80px',
      textAlign: 'center',
    },
    ...root({
      inline: true,
    }),
  },
  itemHead: {
    ...item({
      gridArea: 'A / A / C / C',
      alignSelf: 'center',
      justifySelf: 'center',
    }),
  },
  itemHeadBorder: {
    ...item({
      gridArea: 'A / A / C / C',
    }),
  },
  itemBody: {
    ...item({
      gridArea: 'D / D / G / G',
      alignSelf: 'center',
      justifySelf: 'center',
    }),
  },
  itemBodyBorder: {
    ...item({
      gridArea: 'D / D / G / G',
    }),
  },
  itemSide: {
    ...item({
      gridArea: 'E / E / I / I',
      alignSelf: 'center',
      justifySelf: 'center',
    }),
  },
  itemSideBorder: {
    ...item({
      gridArea: 'E / E / I / I',
    }),
  },
}))({});
return (
  <div className={classes.root}>
    <span className={classes.itemHead}>head</span>
    <span className={classes.itemBody}>body</span>
    <span className={classes.itemSide}>side</span>
    <div className={classes.itemHeadBorder}></div>
    <div className={classes.itemBodyBorder}></div>
    <div className={classes.itemSideBorder}></div>
  </div>
);

居中或者其他布局

使用 justifySelf alignSelf 来对元素进行布局，可以节省使用定位元素带来的额外的性能开销与心智负担

headbodyside
const template = `
  "A   B    C"  80px
  "D   E    F"  80px
  "G   H    I"  80px /
  100px 100px 100px
`;
const [align, setAlign] = React.useState('center');
const [justify, setJustify] = React.useState('center');
const [root, item] = makeBundled({
  grid: template,
});
const classes = makeStyles(() => ({
  root: {
    '& > div': {
      boxShadow: '0 0 0 1px #000 inset',
      lineHeight: '80px',
      textAlign: 'center',
    },
    ...root({
      inline: true,
    }),
  },
  itemHead: {
    ...item({
      gridArea: 'A / A / C / C',
      alignSelf: ({align}) => align,
      justifySelf: ({justify}) => justify,
    }),
  },
  itemHeadBorder: {
    ...item({
      gridArea: 'A / A / C / C',
    }),
  },
  itemBody: {
    ...item({
      gridArea: 'D / D / G / G',
      alignSelf: ({align}) => align,
      justifySelf: ({justify}) => justify,
    }),
  },
  itemBodyBorder: {
    ...item({
      gridArea: 'D / D / G / G',
    }),
  },
  itemSide: {
    ...item({
      gridArea: 'E / E / I / I',
      alignSelf: ({align}) => align,
      justifySelf: ({justify}) => justify,
    }),
  },
  itemSideBorder: {
    ...item({
      gridArea: 'E / E / I / I',
    }),
  },
}))({align, justify});
return (
  <>
    <Box mb={3}>
      <Button onClick={() => setAlign('start')}>alignSelf: start</Button>
      <Button onClick={() => setAlign('center')}>alignSelf: center</Button>
      <Button onClick={() => setAlign('end')}>alignSelf: end</Button>
      <Button onClick={() => setJustify('start')}>justifySelf: start</Button>
      <Button onClick={() => setJustify('center')}>justifySelf: center</Button>
      <Button onClick={() => setJustify('end')}>justifySelf: end</Button>
    </Box>
    <div className={classes.root}>
      <span className={classes.itemHead}>head</span>
      <span className={classes.itemBody}>body</span>
      <span className={classes.itemSide}>side</span>
      <div className={classes.itemHeadBorder}></div>
      <div className={classes.itemBodyBorder}></div>
      <div className={classes.itemSideBorder}></div>
    </div>
  </>
);

特殊单位

当需要响应式布局的时候，就需要使用到特殊单位 fr 与 auto, fr 代表当前剩余空间， 计算方式则是当前宽度，减去其他元素已经使用的宽度，然后与其他的 fr 元素分享剩余的宽度

auto 则是使用当前元素的最小尺寸(需要在 Mdn 上确认下细节)

headbodyside
const template1 = `
  "A    B   C"  80px
  "D    E   F"  1fr
  "G    H   I"  1fr /
  100px 1fr 1fr
`;
const template2 = `
  "A    B   C"  auto
  "D    E   F"  1fr
  "G    H   I"  1fr /
  auto 1fr 1fr
`;
const [align, setAlign] = React.useState('center');
const [justify, setJustify] = React.useState('center');
const [currentTemplate, setCurrentTemplate] = React.useState(template1);
const [root, item] = makeBundled({
  grid: ({template}) => template,
});
const classes = makeStyles(() => ({
  root: {
    '& > div': {
      boxShadow: '0 0 0 1px #000 inset',
      lineHeight: '80px',
      textAlign: 'center',
    },
    height: 300,
    width: 400,
    ...root({
      inline: true,
    }),
  },
  itemHead: {
    ...item({
      gridArea: 'A / A / C / C',
      alignSelf: 'center',
      justifySelf: 'center',
    }),
  },
  itemHeadBorder: {
    ...item({
      gridArea: 'A / A / C / C',
    }),
  },
  itemBody: {
    ...item({
      gridArea: 'D / D / G / G',
      alignSelf: 'center',
      justifySelf: 'center',
    }),
  },
  itemBodyBorder: {
    ...item({
      gridArea: 'D / D / G / G',
    }),
  },
  itemSide: {
    ...item({
      gridArea: 'E / E / I / I',
      alignSelf: 'center',
      justifySelf: 'center',
    }),
  },
  itemSideBorder: {
    ...item({
      gridArea: 'E / E / I / I',
    }),
  },
}))({template: currentTemplate});
return (
  <>
    <Box mb={3}>
      <Button onClick={() => setCurrentTemplate(template1)}>fr template</Button>
      <Button onClick={() => setCurrentTemplate(template2)}>auto template</Button>
    </Box>
    <div className={classes.root}>
      <span className={classes.itemHead}>head</span>
      <span className={classes.itemBody}>body</span>
      <span className={classes.itemSide}>side</span>
      <div className={classes.itemHeadBorder}></div>
      <div className={classes.itemBodyBorder}></div>
      <div className={classes.itemSideBorder}></div>
    </div>
  </>
);
